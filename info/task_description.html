Enunciado

Ana, uma assistente editorial, está trabalhando na revisão de um livro e precisa combinar várias frases armazenadas em uma lista em um único parágrafo. Ela quer uma ferramenta que ajude a concatenar todas as frases em uma única string, com cada frase separada por um espaço. Escreva uma função que receba uma lista de strings (frases) e retorne uma única string que é a concatenação de todas as frases na lista, separadas por um espaço, usando um loop for.

Entrada: Uma lista de strings.

Saída: Uma única string que é a concatenação de todas as frases da lista, separadas por um espaço.
