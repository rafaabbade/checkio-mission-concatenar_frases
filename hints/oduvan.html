Como posso usar um loop for para passar por todos os elementos de uma lista?

Você pode usar um loop for para iterar por todos os elementos de uma lista da seguinte forma: Copie e cole o código abaixo no site online-python.com 

palavras = ["Python", "é", "ótimo"]
for palavra in palavras:
    print(palavra)
A vantagem de usar o for é que você pode processar cada elemento da lista independentemente do tamanho da lista.

Como concatenar strings em Python usando um loop for?

Para concatenar strings em Python, você pode usar um loop for para adicionar cada string a uma string acumuladora, da mesma forma que fazer com números. A vantagem de usar o for é que você pode concatenar as frases sem saber a quantidade de frases antecipadamente.

Existe uma forma mais eficiente de concatenar várias strings em Python?

Sim, você pode usar o método join para concatenar todas as strings em uma lista de forma eficiente. Exemplo: 

frases = ["Esta é a primeira frase.", "Aqui está a segunda.", "E esta é a terceira."]
paragrafo = "".join(frases)
print(paragrafo)  # Resultado: "Esta é a primeira frase. Aqui está a segunda. E esta é a terceira."
